# Classe Pilha (Stack)
- Definição

 Pilhas são estruturas de dados onde os items são organizados (empilhados) um por cima do outro, o ultimo a entrar
é o primeiro a sair.

Comportamento de pilha

LIFO

Last in, firt out

- Verificar quantidade de elementos na pilha

- Empilhar elemento

- Verificar se pilha está vazia

- Espiar/Verificar elemento topo da pilha

- Desempilhar elemento da pilhar
